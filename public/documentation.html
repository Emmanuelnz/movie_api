Documentation

Objective: Build the server-side component of a “movies” web application. The application will give users information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

POST Register a new user /users A JSON with data about the newly added user (Name, favoriteMovies, id). Ex.
{
"_id": "62d08e717e75c65b52db36b2",
"username": "sample name",
"password": "sample password",
"email": "sample email",
"Birthday": "sample birthdate",
"favoriteMovies": [users favorite movies go in here]
PUT Updates user info (ex. username) /users/{Username} A JSON with data about the users updated info (id, updated Username,Password, email, birthdate, favoriteMovies list). Ex.
{
"_id": "sample id",
"username": "sample name",
"password": "sample password",
"email": "sample password",
"Birthday": "sample birthdate",
"favoriteMovies": []
}
POST Adds movie to user's favorite movie list /users/{Username}/Movies/{MovieID} A JSON with the movie's ID in the user's favoriteMovies Array/List. Ex.
{
"_id": "sample id",
"username": "sample name",
"password": "sample password",
"email": "sample password",
"Birthday": "sample birthdate",
"favoriteMovies": [MovieID would go here]
}
DELETE Removes movie from user's favorite movie list /users/{Username}/Movies/{MovieID} A JSON with the updated favoriteMovies list Ex.
{
"_id": "sample id",
"username": "sample name",
