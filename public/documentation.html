Documentation Page

Objective: Build the server-side component of a “movies” web application. The application will give users information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

HTTP Method Request URL Response
GET Gets a list of all movies /movies A JSON with data about all the movies
GET Gets data about a single movie by title (ex. "The Batman") /movies/{title} A JSON with data about a single movie(Data- Movie Title, Description, Directors,Genre)
Ex.{
Title: 'Batman vs Robin',
Description: "Damian Wayne has a hard time accepting his father's no-killing rule, and soon starts to believe his destiny lies within a secret society.",
Directors: {
Name: 'Jay Oliva',
Bio: 'Jay Oliva is an American storyboard artist, film producer and animated film director working at Lex and Otis animation studio.',
Born: '1976, USA',
Death: 'Still Alive' },
Genre: {
Name: 'Action',
Description: 'The action genre are fast-paced and include a lot of action like fight scenes, chase scenes, and slow-motion shots. They can feature superheroes, martial arts, or exciting stunts. These high-octane films are more about the execution of the plot rather than the plot itself.' }
}
GET Gets data about a genre by genre name (ex."Action") /movies/{genre}/{genre name} A JSON with data about a genre (data- Genre Name, Description)
Ex.Genre: {
Name: 'Action',
Description: 'The action genre are fast-paced and include a lot of action like fight scenes, chase scenes, and slow-motion shots. They can feature superheroes, martial arts, or exciting stunts. These high-octane films are more about the execution of the plot rather than the plot itself.' }
GET Gets data about a director by name (ex. "Sam Raimi") /movies/{directors}/{director name} A JSON with data about the directors (Data- Name,Short Bio, Birth,Death)
Directors: {
Name: 'Jay Oliva',
Bio: 'Jay Oliva is an American storyboard artist, film producer and animated film director working at Lex and Otis animation studio.',
Born: '1976, USA',
Death: 'Still Alive' }
POST Register a new user /users A JSON with data about the newly added user (Data-Name, favoriteMovies, id) {
"name": "Leo",
"favoriteMovies": [{empty array/list}],
"id": "b030cd61-5755-41b4-ace8-1f037cca818c"
}
PUT Updates user info (ex. username) /users/{name} A JSON with data about the users updated info (id, updated Name, favoriteMovies list) {
"id": 1,
"name": "Sam fisher",
"favoriteMovies": [{empty array/list}]
}
POST Adds movie to user's favorite movie list /users/{id}/{movie title} Message stating "{movieTitle} has been added to user {id}'s favoriteMovies list"
DELETE Removes movie from user's favorite movie list /users/{id}/{movie title} Message stating "{movieTitle} has been removed from user {id}'s favoriteMovies list"
DELETE Delete account/deregister current user /users/{id} Message stating "User has been deleted"
